## Version History

### v2.2.0 (November 28, 2025) - CURRENT

**Major Feature Release - Advanced Editing Tools**

New Features:
- **Curves Adjustment Panel** - Professional RGB channel curves editing
  - Individual R, G, B channel control
  - Presets: Linear, S-Curve, Fade, High Key
  - Interactive drag-and-drop control points
- **Split Toning Panel** - Color grading for highlights and shadows
  - Hue/Saturation controls for highlights and shadows
  - Balance slider
  - Presets: Cinematic, Vintage, Teal and Orange, Cool Fade, Warm Sunset
- **Custom Filter Creator** - Save and reuse your own filters
  - Save current adjustments as custom filter
  - Apply saved filters with one tap
  - JSON serialization for persistence
- **Batch Processing Screen** - Edit multiple photos at once
  - Multi-image selection
  - Apply same filter/adjustments to all
  - Progress tracking
- **Social Sharing Service** - Share to multiple platforms
  - Instagram, Facebook, Twitter, Snapchat
  - WhatsApp, Telegram, Native share sheet
  - Auto-generated hashtags
- **PWA Support** - Installable web app
  - Web app manifest with shortcuts
  - Share target for receiving images
  - Offline-ready architecture

### v2.1.0 (November 27, 2025)
